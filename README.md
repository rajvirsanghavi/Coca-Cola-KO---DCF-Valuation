# Coca-Cola (KO) — DCF Valuation Model

A Discounted Cash Flow valuation of Coca-Cola built in Microsoft Excel.

## What this project covers
- Historical financials: Revenue & Free Cash Flow (2022–2024)
- 5-year FCF projections (2025E–2029E)
- Terminal value calculation
- Intrinsic value per share estimation
- Sensitivity analysis on WACC assumptions

## Key Findings
- Intrinsic Value (DCF): $47.77 per share
- Current Market Price: ~$81.63
- Market Premium: ~41% above DCF value
- Conclusion: Premium reflects KO's brand moat & dividend reliability

## Assumptions Used
- Revenue Growth Rate: 5%
- Normalized FCF Margin: 21%
- WACC: 8%
- Terminal Growth Rate: 2.5%

## Tools Used
Microsoft Excel | DCF Methodology | Macrotrends Data
